I started learning rust programming language for fun. There is a lot to it, especially when all I know is bash scripting and python.

The general consensus online is that it isnt a very beginner friendly language, but I'm managing ok so far.

what really drew me into Rust, compared to other languages like C or C++, is that Rust is memory safe by default. Where as in C and other languages you need to manually manage memory which can lead to memory leaks etc.


One of the ways that rust makes it so "easy" to write safe code, is that variables are immutable (unchangeable) by default and also there is ownership and borrowing rules for accessing them outside of their scope (functions)

this has been a lot to wrap my head around tbh. Also the differences of expressions and statments confused me for a bit, but once you get the hang of it it's pretty rewarding.
