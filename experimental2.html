ABOUT:

This experiment was to try to auto-update my website using a bash script to edit my .html file

first the shell script attempts to curl a random word from the internet as well as stores the time in a variable.

Next it appends these variables into my .html file with properly formatted html, as well as deletes the last chunk so it's only ever 7 days long.

The formated html is as follows

<div class="round3">
<p> \ ${timestamp}</p>
<p>random word: \ ${word}</p>
</div>

Then it pulls line 99 of the html file and +1 to increase the counter, deletes the old line 99 and inserts the new variable

Finally my script will git add/commit/push the new file to my websites repo

This took some fiddling to properly automate. Trying to run from cron.daily would try running it as root, so I'd get git permission errors when trying to git push. The solution was to specify at the start of the bash script to run the following commands as my username instead of root. ezpz
