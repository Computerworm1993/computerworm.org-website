Essential Shell Commands
some of my favourites

December 19, 2025


netcat: transfering large files over networks using TCP or UDP

on server, listen for file on specified port

on source device. Cmpress with tar and send it over ssh connection

ls | grep:combined list and search

learning how to pipe into grep has been so insanely useful for finding files through the terminal

df -h: read disk usage; in -h "human readable" format

This has been useful as I fill up my servers drives with media, movies and music

systemctl commands: checking status, stopping, starting and enabling services

such as vpn, tailscaled, firewalld, jellyfin. Very useful as i frequently open and close ports for netcat, and switch between vpn services depending on what I'm doing

locate a quicker 'find' command

quite useful for finding the path to files and directories. When I experiment a little too hard and need a fresh install. After uninstalling software I can locate it to see what files got left behind like config files or whatever to delete so I can start completely clean and try again, till I break it next anyways lol. Though 'dnf purge' usually gets them all, but I always forget to use it and use 'dnf remove' instead by force of habit lol
