worm-org.sh


This is my script for quickly and easily updating the "STATUS" that moves across the top of my websites homepage.


No real new techniques used in this one, it's sort of a combo of my zsh_history.sh and my astro_scrape.sh. Editing html and git pushing automatically.

it has been very handy for me though. Its nice to be able to just do a quick status update right from my terminal.


Additionally I included an argument (-a) to push everything and not just the page containing the status.
Considering how often i tweak this site, instead of having to type out

git add .
git commit -m "message"
git push

I can just type out 1 line.
worm-org.sh -a

which doesn't seem like all that less to type but it adds up.



I also added within the '-a' argument to push all, a section that curls the github api to get the json of how much total data is stored in my remote repository. It returns a value in KB so I /1000 to get MB, then edit it into my html before pushing

since it can only push after getting the data the value will always be 1 push behind the actual amount. Unless I were to push then get the json, edit the html and do a second push, but that just seems silly to me

anyways its some pretty cool stuff
