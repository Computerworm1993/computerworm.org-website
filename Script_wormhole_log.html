wormholeLog.sh

This is a script I wrote for monitoring traffic to my "netflix-esque" jellyfin server


It works by querying the SQLite database file for the most recent 10 entries.

It then pipes the output of this query into a text file.

Then within a while loop I have repeat every 5 seconds, it performs a second query of the same database. This time piping output into an array. A few if;elsif;else functions are then used to determine if the value of the most recent entry has changed since the last query

If the value has changed. Then I search for usernames within the index of the changed array.
If user=mom then print index in cyan.
If user=dad then print index in green.
If user=worm then print index in yellow
etc. etc.
else print index in white



makes me feel like a peeping tom spying on people lol, but it is useful to see quickly at a glance for troubleshooting and if my network feels like it's being bogged down etc.
